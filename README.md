# Choco - Digital Web Menu
https://choco-menu.vercel.app/
A stylish, modern, and interactive digital menu created for "Choco", a coffee, bar, and snacks establishment. This project is built with standard web technologies and is designed to be easily viewed on any device, from desktops to mobile phones.


## Features

-   **Clean & Modern Design:** Aesthetically pleasing layout with a focus on readability and brand identity.
-   **Interactive Tab Navigation:** Users can easily switch between categories (Coffees, Drinks, Snacks) with smooth animations, without needing to scroll through the entire menu.
-   **Dynamic Backgrounds:** Each menu category features a unique, faded background image that enhances the theme without distracting from the content.
-   **Fully Responsive:** The layout is optimized for a great viewing experience on both large desktop screens and small mobile devices.
-   **Well-Structured Code:** The project is organized into separate HTML, CSS, and JavaScript files, following best practices for maintainability.
-   **Custom Typography:** Utilizes the "Lora" font from Google Fonts for an elegant and classic feel.

## Customization

This project is designed to be easily customized:

-   **Menu Items:** To add, remove, or edit menu items and prices, modify the `<section>` elements inside `Choco.html`.
-   **Styling (Colors, Fonts):** All visual styles can be changed in the `choco.css` file.
-   **Background Images:** To change the background image for a category, find its ID in `choco.css` (e.g., `#cafes`) and replace the `url(...)` with a link to your new image.
-   **Interactive Logic:** The JavaScript that controls the tab switching is located in `choco.js`.

## Technologies Used

-   **HTML5:** For the structure and content of the menu.
-   **CSS3:** For all styling, layout (Flexbox), animations, and responsive design (`@media` queries).
-   **JavaScript (ES6):** For the interactive tab functionality.
